Payroll Computation Using Functions

Course: COPROG1

Requirements

  • Create a program to compute an employee's monthly salary.
  • User inputs hours worked per week (4 weeks per month).
  • Save as finalCode-<section>-<GroupNo>.java (e.g., finalCode-c1a-GRP1.java).
  • Use functions in your solution.

Computation & Deductions

  • Regular: 40 hours/week. Overtime: >40 hours/week (paid 125% rate).
  • Gross pay = (rate × regular hours) + overtime pay.
  • Deductions: SSS, Pag-IBIG, PhilHealth.

SSS Contribution

Gross Pay (GP)SSS Computation
GP ≤ 5,000Php 105.00
5,001 – 10,0005% of GP

> 15,00012% of GP + P110

Pag-IBIG Contribution

  • GP < 5,000: P100.00
  • GP > 5,000: 3% of GP

PhilHealth Contribution

  • P120.00 (if hours worked ≥ 10)
  • 0 (if hours worked < 10)

Tax Computation

Other Allowances & Deductions

  • Manager: P5,000/week transport allowance.
  • Dependent: Deduct P1,000 for each dependent.
  • Net salary = gross pay – total deductions.

Employee Info & Output

  • Input: Last Name, First Name, Middle Name, Department, Position, Rate/hour, Hours worked.
  • Output: Pay slip with info, salary details, gross pay, net pay.
  • Option to run again or exit.

Required Functions

  • computeGP – gross pay
  • computeDed – total deductions
  • computeNP – net pay
  • All must return a value.

Documentation

  • Description of the program
  • Program design: IPO chart, flowchart
  • Inventory: variables/functions with descriptions
  • Sample output (screenshots)
  • Save as finalDocu-<section>-<GroupNo> (PDF for Canvas)

Grading Criteria

  • Documentation – 30 pts (Flowchart 15, Completeness 10, Organization 5)
  • Program – 50 pts (Functionality 25, Requirements 10, Functions 15)
  • Total: 80 pts
