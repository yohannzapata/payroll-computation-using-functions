Project Setup

Simplified steps to get the Java payroll project running.

Home Collaboration

Steps

  • 1. Install & configure Git
    • Create a GitHub account: github.com.
    • Install Git: Download for Windows.
    • Open PowerShell and set your name & email:
    git config --global user.name "Your Name"Tell Git your name. git config --global user.email "you@example.com"Tell Git your email.

    Check:

    git config --listShow your Git settings. git --versionShow your Git version.
  • 2. Install Java (JDK 17+)
    • Download Temurin 17: link.
    • Keep defaults (add to PATH, set JAVA_HOME).
    • Reopen PowerShell; verify:
    java -versionCheck your Java version. javac -versionCheck your Java compiler version.
  • 3. Install VS Code
    • Download: VS Code.
    • Extensions: Extension Pack for Java; (optional) GitLens.
    • Increase font size in Settings if needed.
  • 4. Clone the repository
    • Go to a folder (e.g. Documents).
    • Run:
    cd $env:USERPROFILE\DocumentsGo to your Documents folder. git clone https://github.com/yohannzapata/payroll-computation-using-functions.gitCopy the project to your computer. cd payroll-computation-using-functionsGo into the project folder. code .Open the project in VS Code.
  • 5. Compile & run
    • Open Main.java.
    • In terminal:
    javac Main.javaTurn your code into a program. java MainRun your program.

    Fix any syntax errors, then re-run.
